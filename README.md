# Laberinto26-Python

Solo se encuentra funcional el juego por consola, el juego con intefaz no es 
completamente funcional pero sí ejecutable.

Si se quiere jugar con la consola se debe ejecutar el archivo mainFile en src 
y la variable global "consola" de mainFile debe ser True.

Si se quiere jugar con la GUI se debe ejecutar el archivo GUI con la variable 
"consola" de mainFile en False.

Si se quiere modificar el mapa, se debe cambiar el JSON que se encuentra en la
carptera laberintos dentro de src.

Existe un log del juego en juego.log que se actualizará en cada partida.

En LaberintoBuilderTest se encuentran los test del laberinto.
